Warung Mie Instan (WARMINTAN)

There several big rules in our business, that will make our business grow smoothly:

1. The only one that can enter warehouse only the chef
2. The only one that will take order is the waiter
3. The only one that will cook of course only chef, and chef will get the order from the waiter
4. The only way to add stock is from waiter > chef > warehouse
5. Waiter will input any command through CLI
